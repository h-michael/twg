# twg

twg is twitter client for cli that written by golang.

## Usage

```
$ go get https://github.com/waka/twg
$ twg
```

Homebrew

```
$ brew install twg
```

## Key binds

`hjkl` - left/down/up/right
`r`    - refresh

## Build in local machine

resolve dependencies using [dep](https://github.com/golang/dep).

```
$ make deps
```

build and run.

```
$ make
$ bin/twg
```
